PyTeX Lib

A Python library for inserting function outputs into LaTeX documents

Home | Projects | GitHub

Overview

PyTeX Lib allows Python functions to dynamically write results into LaTeX documents.

Use the @write_to_latex decorator to insert values into LaTeX files at specified placeholders.

Installation

pip install pytex-lib
