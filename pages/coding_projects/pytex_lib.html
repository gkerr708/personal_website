PyTeX Lib

A Python library to link your functions to LaTeX documents

Home | Projects | GitHub

Overview

PyTeX Lib allows Python functions to dynamically write results into LaTeX documents.

Use the @write_to_latex decorator to insert values into LaTeX files at specified placeholders.

Installation

pip install pytex-lib

Usage

Import the Library

from pytex_lib import write_to_latex

Define a Function with the Decorator

@write_to_latex
def compute_square(x):
    return f"The square of {x} is {x**2}"

LaTeX Document Before Execution

\documentclass{article}
\begin{document}

Here is the computed result:
% RESULT_PLACEHOLDER

\end{document}

Run the Function

file_path = "document.tex"
keyword = "RESULT_PLACEHOLDER"
compute_square(4, file_path=file_path, keyword=keyword)

LaTeX Document After Execution

\documentclass{article}
\begin{document}

Here is the computed result:
% RESULT_PLACEHOLDER
The square of 4 is 16

\end{document}
