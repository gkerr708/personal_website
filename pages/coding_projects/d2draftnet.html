Project Overview

D2DraftNet is a machine learning model designed to predict the outcome of Dota 2 matches based on hero drafts. The model utilizes embeddings to transform a hero draft into a vector representation in an abstract embedding space, which is then processed by a deep neural network (DNN) for training and inference.

How It Works

  • The draft of each team (Radiant & Dire) is represented as a binary vector.
  • This vector is passed through an embedding layer, which maps each hero pick to a low-dimensional numerical representation.
  • A Deep Neural Network (DNN) processes this embedding, learning complex interactions between hero picks.
  • The model outputs a win probability for the Radiant team.
