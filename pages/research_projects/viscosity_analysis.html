Viscosity Analysis

A Python-based framework for analyzing viscosity data from the TRIOS software for the TA Instruments HR10 Rheometer

Home | Projects | GitHub

Overview

Viscosity Analysis is a Python framework designed to process and analyze viscosity data obtained from the TRIOS software for the TA Instruments HR10 Rheometer. This project automates the extraction, filtering, and visualization of rheological data to study the viscosity behavior of complex fluids.

The framework supports data-driven analysis of shear-dependent viscosity, allowing researchers to model experimental results using rheological equations like the Cross model. It enables comparison of different experimental conditions, such as variations in solute concentration, solvent composition, and applied shear rates.

The project also includes automated plotting and data fitting routines to extract key rheological parameters, making it easier to evaluate trends in non-Newtonian fluid behavior.

How It Works

The framework follows a structured approach to process viscosity data:

  • Data Extraction: The software loads CSV files exported from TRIOS and extracts relevant experimental data.
  • Data Filtering: Users can specify conditions to filter experiments based on composition, date, and experimental parameters.
  • Rheological Modeling: The extracted data is fitted to models like the Cross equation to determine key viscosity parameters.
  • Visualization: The framework generates log-log plots to visualize shear-dependent viscosity behavior.
  • Parameter Analysis: Key rheological properties such as zero-shear viscosity, infinite-shear viscosity, and flow consistency index are extracted for comparison across different experiments.
